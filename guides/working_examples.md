# Working Examples

Examples exercise JSONL persistence and AITrace export.

## What This Covers

The examples are local and do not require a live model.

## Worked Example

```bash
mix run examples/trace_jsonl_mock.exs
mix run examples/aitrace_export_live.exs
```
